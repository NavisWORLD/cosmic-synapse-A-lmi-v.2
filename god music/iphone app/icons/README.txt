ICON FILES NEEDED

Place the following icon files in this directory:

Required Icons:
- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- apple-touch-icon.png (180x180)
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

How to Create:
1. Open create-icons.html in a browser (in parent folder)
2. Click "Generate All Icons"
3. Click "Download All Icons"
4. Save the files to this icons/ folder

Or use online tools:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator

The app will work without icons, but they won't appear on the home screen
until you add them here.
